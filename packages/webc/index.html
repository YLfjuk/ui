Documentation Example

Introduction

This is an example of how the headings tree works. Notice how it creates a navigation structure based on the document's headings.

Getting Started

The headings tree automatically detects and displays all heading elements in your document.

Prerequisites

Make sure your headings have proper hierarchy and meaningful content.

Features

The headings tree supports multiple levels of nesting and automatically adjusts the indentation based on heading levels.

Styling Options

The component uses Tailwind CSS classes for styling, making it easy to modify its appearance.
